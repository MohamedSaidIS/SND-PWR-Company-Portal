# SND Power Employee Tracker (HR App) 

Developed an HR and Employee Management application for SND Power to streamline employee services and internal support operations. The app integrates with Microsoft 365 for secure authentication and access to company systems.

Key Features (ordered by importance):

Support Case Management System 
Integrated with Microsoft Dynamics 365 Supply Chain Management and Dynamics 365 Commerce to enable employees to create, track, and manage support cases (IssuesTracker). Users can add comments and upload images to help resolve issues efficiently.

KPI Performance Tracker 
A visual dashboard showing daily, weekly, and monthly KPIs to help employees monitor and improve their performance.

HR Services (Vacation Management) 
Allows employees to submit vacation requests and view their vacation balance directly through the app.

Real-Time Notifications 
Implemented a notification system to alert users about case updates, new comments, KPI updates, and vacation request approvals or rejections, ensuring employees stay informed about important activities.

Microsoft Services Integration 
Enables users to open and access relevant Microsoft applications directly from the app for a seamless workflow.

Authentication with Microsoft 365 
Secure single sign-on using Microsoft 365 accounts for convenient and safe access.

Employee & Manager Information 
Displays personal employee details and manager information for quick access and communication.

Responsive and User-Friendly Interface 
Designed to provide a smooth experience across different devices and screen sizes.
